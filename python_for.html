파이썬 For 문 완전 정복

for문 하나로 100번, 1000번도 처리할 수 있습니다.

for문이란?

반복 횟수가 정해져 있을 때 사용합니다.
예: 출석부 30명 이름 읽기, 유튜브 재생목록 10곡 재생

for문의 핵심 구조

변수 i가 목록에서 값을 하나씩 꺼냅니다.

for i in [목록]:
    실행할 코드

3. 숫자 범위 다루기: range()

range(시작, 끝, 간격)을 사용합니다. '끝' 숫자는 포함되지 않는다는 점이 핵심입니다. 

# 1부터 10까지 2씩 건너뛰며 출력 (1, 3, 5, 7, 9)
for i in range(1, 11, 2):
    print(i, end=" ")

print() # 줄바꿈

# 5부터 1까지 거꾸로 카운트다운 (간격이 음수)
for i in range(5, 0, -1):
    print(i, end=" ")

4. 누적 합계

for문으로 숫자를 하나씩 더해서 합계를 구할 수 있습니다. 합계를 담을 변수를 먼저 0으로 준비하는 게 핵심입니다.

total = 0             # 합계를 담을 변수, 반드시 0으로 시작

for i in range(1, 6):  # 1, 2, 3, 4, 5
    total = total + i  # 매번 i를 더해줌

print(total)          # 15
15

5. 별 찍기

for문과 print()를 조합하면 다양한 패턴을 만들 수 있습니다. '*' * n은 별을 n개 출력합니다.

# 계단 모양 별 찍기
for i in range(1, 6):
    print('*' * i)
*
**
***
****
*****

Example — 역삼각형 별 찍기 원리 이해

계단 모양 별은 range(1, 6)으로 i가 1→5로 증가했습니다.
역삼각형은 반대로 i가 감소해야 합니다. range()의 세 번째 인자를 음수로 사용합니다.

🔍 핵심 비교

모양 range() i의 변화 출력
계단 ▲ range(1, 6) 1 → 2 → 3 → 4 → 5 * → ** → *** → ****→ *****
역삼각형 ▽ range(5, 0, -1) 5 → 4 → 3 → 2 → 1 ***** → **** → *** → ** → *

실행 결과를 확인하고, range() 값이 어떻게 바뀌는지 관찰해보세요:

# i가 5부터 1까지 -1씩 줄어듭니다
for i in range(5, 0, -1):
    print(f"i={i} → ", '*' * i)
i=5 → *****
i=4 → ****
i=3 → ***
i=2 → **
i=1 → *

6. Break & Continue

반복 중간에 흐름을 제어할 수 있습니다. break는 반복을 즉시 끝내고, continue는 현재 순서를 건너뜁니다.

# break: 3을 만나면 즉시 반복 종료
for i in range(1, 6):
    if i == 3:
        break
    print(i, end=" ")

print()

# continue: 3을 건너뛰고 계속 진행
for i in range(1, 6):
    if i == 3:
        continue
    print(i, end=" ")
1 2
1 2 4 5

Example — break 동작 원리 이해

break는 반복문을 즉시 탈출시킵니다. 원하는 것을 찾았을 때 더 이상 탐색을 계속할 필요가 없을 때 사용합니다.

🔍 break가 없을 때 vs 있을 때

❌ break 없이 전부 탐색

시도: 1234 ❌
시도: abcd ❌
시도: secret ✅
시도: hello ← 불필요한 탐색!

✅ break로 즉시 탈출

시도: 1234 ❌
시도: abcd ❌
시도: secret ✅
→ 즉시 반복 종료!

코드에서 break가 어느 시점에 실행되는지 확인해보세요:

passwords = ['1234', 'abcd', 'secret', 'hello']
answer = 'secret'

for pw in passwords:
    print('시도:', pw)
    if pw == answer:
        print('✅ 비밀번호 일치! 탐색 종료')
        break               # ← 여기서 반복문 탈출
print('반복문 이후 코드')   # break 후에도 이 줄은 실행됨
시도: 1234
시도: abcd
시도: secret
✅ 비밀번호 일치! 탐색 종료
