1. For 문의 기본 구조

가장 많이 사용하는 range() 함수로 숫자를 반복하는 기본 형태입니다.

for i in range(3):
    print("반복 중입니다. 현재 숫자:", i)

# 주의: 반복문이 끝나야 이 코드가 실행됩니다. (들여쓰기 없음)
print("끝!")

2. 리스트와 문자열 반복

숫자뿐만 아니라 글자(String)나 목록(List)의 내용도 하나씩 꺼낼 수 있습니다.

# 1. 문자열 반복: 글자 하나씩 꺼내기
for i in "Pizza":
    print(i, end="-")
print() # 줄바꿈

# 2. 리스트 반복: 목록 아이템 하나씩 꺼내기
cart = ['사과', '우유', '콩나물']
for item in cart:
    print("장바구니 확인:", item)

3. 숫자 범위 다루기: range()

range(시작, 끝, 간격)을 사용합니다. '끝' 숫자는 포함되지 않는다는 점이 핵심입니다. 

# 1부터 10까지 2씩 건너뛰며 출력 (1, 3, 5, 7, 9)
for i in range(1, 11, 2):
    print(i, end=" ")

print() # 줄바꿈

# 5부터 1까지 거꾸로 카운트다운 (간격이 음수)
for i in range(5, 0, -1):
    print(i, end=" ")

4. 반복문 안에 반복문 (중첩 루프)

시계의 '분침'이 한 바퀴 돌 때마다 '시침'이 한 칸 움직이는 원리와 같습니다. 구조를 잘 살펴봅시다.

for i in range(2, 5):      # i는 '단' (2, 3, 4)
    print(f"--- {i}단 ---")

    for j in range(1, 10): # j는 '곱하는 수' (1~9)
        print(f"{i} x {j} = {i*j}")

✏️ [중간 점검] 구구단 퀴즈

위 코드에서 print 문은 총 몇 번 실행될까요?

5. 별 찍기 집중 공략 ⭐

논리적 사고력을 키우는 최고의 훈련입니다. i가 변할 때 별의 개수가 어떻게 변하는지 확인하세요.

6. 흐름 제어 (Break & Continue)

반복문을 자유자재로 다루기 위한 키워드입니다.

  • break: 반복문을 박차고 나갑니다. (완전 종료)
  • continue: 이번 순서만 건너뛰고 다음 순서로 갑니다.
for i in range(1, 10):
    if i == 3:
        continue  # 3은 건너뜀
    if i == 6:
        break     # 6이 되면 종료
    print(i, end=" ")

🚀 실전 코딩 챌린지

퀴즈로 몸을 풀었으니, 이제 진짜 코드를 작성해 볼까요?

Mission 1
월드컵 개최 연도 구하기

2002년부터 2050년까지 4년마다 열리는 월드컵 개최 연도를 출력하고,
총 몇 번 개최되는지 횟수(count)를 세어 마지막에 출력하는 프로그램을 작성하세요.

2002 2006 2010 ... 2050
총 개최 횟수: 13회
정답 코드 확인하기 
count = 0
for year in range(2002, 2051, 4):
    print(year, end=" ")
    count += 1

print(f"\n총 개최 횟수: {count}회")
Mission 2
1부터 100까지 짝수의 합

1부터 100까지의 숫자 중에서 짝수(2, 4, 6...)들만 모두 더한 합계를 구하세요.
(Hint: range의 간격을 이용하거나, if문과 %(나머지) 연산자를 사용하세요.)

1부터 100까지 짝수의 합: 2550
정답 코드 확인하기 
total = 0

# 방법 1: range 간격 활용
for i in range(2, 101, 2):
    total += i

print(f"1부터 100까지 짝수의 합: {total}")
Mission 3
입력받은 숫자의 약수 구하기

사용자에게 정수 n을 입력받아, 그 수의 약수(나누어떨어지는 수)를 모두 출력하는 프로그램을 작성하세요.

숫자를 입력하세요: 12
1 2 3 4 6 12
정답 코드 확인하기 
n = int(input("숫자를 입력하세요: "))

for i in range(1, n + 1):
    if n % i == 0:  # 나머지가 0이면 약수
        print(i, end=" ")
Mission 4
원하는 범위 출력하기

사용자에게 두 개의 숫자(시작, 끝)를 입력받아, 그 사이의 모든 숫자를 출력하는 프로그램을 작성하세요.
(단, 끝 숫자도 포함해서 출력해야 합니다!)

시작 숫자: 3
끝 숫자: 7
결과: 3 4 5 6 7
정답 코드 확인하기 
start = int(input("시작 숫자: "))
end = int(input("끝 숫자: "))

# 끝 숫자까지 포함하려면 end + 1
for i in range(start, end + 1):
    print(i, end=" ")
