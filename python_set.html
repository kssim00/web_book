1. 세트의 기본 구조와 절대 규칙

세트는 중괄호 {}를 사용하여 값을 담습니다.
하지만 두 가지 절대 규칙이 있습니다: ① 중복 금지, ② 순서 없음 

# 1. 세트 생성 (중복된 1, 2는 자동으로 사라짐)
nums = {1, 2, 3, 1, 2}

print(nums)
# 결과: {1, 2, 3}

⚠️ [주의] 빈 세트 만들기의 함정

비어있는 세트를 만들 때 my_set = {} 라고 쓰면 될까요?
(나중에 배울 '딕셔너리'라는 친구가 이 기호를 선점했거든요!)

정답 확인하기
❌ {}는 빈 딕셔너리가 됩니다.
✅ 빈 세트는 반드시 set() 함수로 만들어야 합니다.

2. 리스트(List) vs 세트(Set)

비슷해 보이지만 용도가 완전히 다릅니다. 상황에 맞춰 골라 써야 합니다.

📋 리스트 (List)

순서대로 줄을 섭니다. (인덱스 O)
똑같은 사람이 또 줄을 서도 됩니다.

L = [1, 1, 2]
print(L[0]) 
# 1 (순서가 있어서 0번째 가능)
🚫 세트 (Set)

마구잡이로 모여 있습니다. (인덱스 X)
똑같은 사람은 들어올 수 없습니다.

S = {1, 1, 2}
# print(S[0]) -> 에러! 💥
# 순서가 없어서 번호표가 없음

3. 그림으로 이해하는 집합 연산

아직 학교에서 집합을 안 배웠어도 걱정 마세요.
두 그룹(세트)이 겹치는지, 합치는지 그림을 보면 바로 이해됩니다.

교집합 (&) 너랑 나랑 공통으로
좋아하는 것
합집합 (|) 네 거 내 거 모두 다
합친 것 (중복은 뺌)
차집합 (-) 나한테만 있고
너한테는 없는 것 
pizza = {"치즈", "밀가루", "토마토"}
burger = {"치즈", "밀가루", "패티"}

print(pizza & burger) # 교집합: {'치즈', '밀가루'}
print(pizza | burger) # 합집합: {'치즈', '밀가루', '토마토', '패티'}
print(pizza - burger) # 차집합: {'토마토'} (버거엔 없는 피자만의 재료)

4. 데이터 추가 및 삭제 (Methods)

데이터를 넣고 빼는 함수들입니다. 특히 삭제할 때 remove와 discard의 차이를 기억하세요!

① 추가하기

  • add(값): 데이터 하나를 추가합니다.
  • update([값1, 값2]): 여러 개를 한 번에 추가합니다. (리스트를 넣어도 됩니다)

② 삭제하기 (안전 vs 엄격)

⚠️ remove(값)

값이 없으면 에러(Error)를 내며 화를 냅니다.
확실히 있을 때만 쓰세요.

s = {1, 2}
s.remove(3) 
# Error 발생! 💥
🛡️ discard(값)

값이 없으면 그냥 무시하고 넘어갑니다.
있으면 지우고, 없으면 맙니다.

s = {1, 2}
s.discard(3) 
# 아무 일도 안 일어남

③ 유용한 변환 함수: sorted()

세트는 순서가 없어서 정렬할 수 없지만, sorted()를 쓰면 정렬된 리스트로 변환해 줍니다.

s = {3, 1, 2}
result = sorted(s)
print(result) # [1, 2, 3] (리스트로 바뀜!)

Mission 1
숙제 안 해온 사람 찾기 (차집합)

전체 학생 명단과 숙제를 제출한 학생 명단이 있습니다.
차집합(-)을 이용해 숙제를 안 낸 사람을 찾아내고, 가나다순으로 정렬하세요.
all = {'철수', '영희', '민수', '지수'}
submit = {'영희', '민수'}

정답 코드 확인하기 
all_students = {'철수', '영희', '민수', '지수'}
submitted = {'영희', '민수'}

# 전체 - 제출 = 안 낸 사람
not_submitted = all_students - submitted

# 가나다순 정렬 (리스트로 변환됨)
result = sorted(not_submitted)

print(result) # ['지수', '철수']
Mission 2
게임 아이템 정리 (중복 제거)

던전에서 아이템을 마구 주웠더니 가방에 똑같은 게 너무 많습니다.
리스트를 세트로 변환하여 중복을 없애고, 아이템 종류가 몇 개인지 구하세요.
items = ["검", "물약", "검", "방패", "물약"]

정답 코드 확인하기 
items = ["검", "물약", "검", "방패", "물약"]

# 세트로 변환하면 중복 자동 제거
unique_items = set(items)

print(unique_items) # {'검', '물약', '방패'}
print(len(unique_items)) # 3개
Mission 3
로또 번호 생성기 (랜덤 + 중복방지)

1부터 45까지의 숫자 중 6개를 뽑아야 합니다.
랜덤으로 뽑을 때 중복이 나오면 안 되죠? 세트의 성질을 이용해 구현해보세요.
(힌트: 세트의 개수가 6개가 될 때까지 계속 뽑으세요!)

정답 코드 확인하기 
import random

lotto = set()

# 세트 안에 6개가 찰 때까지 무한 반복
while len(lotto) < 6:
    num = random.randint(1, 45)
    lotto.add(num) # 이미 있는 숫자면 알아서 무시됨

print(sorted(lotto)) # 보기 좋게 정렬해서 출력
