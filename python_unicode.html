유니코드와 문자 변환 (ord, chr)

컴퓨터가 문자를 저장하는 방법 — 숫자와 문자를 자유롭게 변환해봅시다.

1. 유니코드(Unicode)란?

컴퓨터는 문자를 숫자로 저장합니다. 아래 코드를 실행하면 어떤 결과가 나올까요?

print(ord('a'))
→
97
print(ord('A'))
→
65
print(ord('0'))
→
48

💡 알 수 있는 것

문자마다 고유한 번호가 있습니다. 이 번호를 아스키(ASCII) 코드라고 부릅니다.
ord()는 문자를 넣으면 그 번호를 돌려주는 함수입니다.

반대로, 번호를 넣으면 문자가 나오는 함수도 있습니다.

print(chr(97))
→
a
print(chr(65))
→
A
print(chr(48))
→
0

💡 알 수 있는 것

chr()는 번호를 넣으면 그에 해당하는 문자를 돌려주는 함수입니다.
ord()와 chr()는 서로 반대 방향으로 동작합니다.

문자
'A'
ord() →
← chr()
번호
65

자주 사용하는 문자들의 번호 범위를 알아두세요.

문자번호 범위종류
'A' ~ 'Z'65 ~ 90대문자 알파벳 26개
'a' ~ 'z'97 ~ 122소문자 알파벳 26개
'0' ~ '9'48 ~ 57숫자 문자 10개
' '32공백(스페이스)
'가'44032한글 첫 글자

2. 문자를 숫자로: ord()

ord()에는 반드시 문자 하나만 넣어야 합니다.

print(ord('A'))   # → 65
print(ord('a'))   # → 97
print(ord('가'))  # → 44032
65 97 44032

💡 대소문자 차이는 항상 32!

ord('a') − ord('A') = 97 − 65 = 32
모든 알파벳 쌍(b−B, z−Z)이 같은 규칙입니다. 이 규칙만 알면 대소문자 변환을 직접 만들 수 있습니다!

3. 숫자를 문자로: chr()

chr()는 번호를 넣으면 그 번호에 해당하는 문자를 돌려줍니다.

print(chr(65))   # → 'A'
print(chr(66))   # → 'B'
print(chr(97))   # → 'a'
A B a

ord()와 chr()를 합쳐서 쓰면 강력해집니다.

# 'A'에서 2칸 뒤의 글자는?
print(chr(ord('A') + 2))  # 65 + 2 = 67 → 'C'
C

4. 알파벳 출력 & 시저 암호

번호가 연속된다는 점을 이용하면 for문으로 알파벳 전체를 출력할 수 있습니다.

# A부터 E까지 출력
for i in range(65, 70):
    print(chr(i), end=" ")
A B C D E

시저 암호는 각 글자를 일정 칸수만큼 밀어서 다른 글자로 바꾸는 고전 암호입니다.
예를 들어 3칸 밀면: A→D, B→E, C→F

# 'ACE'를 3칸 밀면? → 'DFH'
word = "ACE"
for char in word:
    print(chr(ord(char) + 3), end="")
DFH

5-2. 유니코드 범위로 문자 종류 판별하기

번호의 범위를 이용하면 그 문자가 대문자인지, 소문자인지, 숫자인지 알 수 있습니다.

char = 'G'
code = ord(char)

if 65 <= code <= 90:
    print("대문자입니다.")
elif 97 <= code <= 122:
    print("소문자입니다.")
elif 48 <= code <= 57:
    print("숫자 문자입니다.")
대문자입니다.

📖 상황 문제 문제 1
카이사르 암호 만들기

알파벳 대문자로 이루어진 단어와 숫자 하나를 입력받아, 각 글자를 그 숫자만큼 밀어서 암호화된 단어를 출력하세요.
(끝 알파벳은 다시 앞으로 돌아옵니다. 예: Z에서 +1이면 A가 됩니다.)

입력: ACE / 3 → 출력: DFH 입력: PYTHON / 1 → 출력: QZUIPO 입력: XYZ / 2 → 출력: ZAB
💡 힌트
끝 알파벳 처리: chr((ord(char) - 65 + n) % 26 + 65)
65를 빼서 0~25로 만든 뒤, 26으로 나눈 나머지를 이용합니다.
