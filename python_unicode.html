유니코드와 문자 변환 (ord, chr)

"컴퓨터는 숫자로 대화합니다. 문자의 숨겨진 번호를 찾아보세요!"

🆔 ord(문자)

주민등록증 검사기
문자를 넣으면 고유 번호(숫자)를 알려줍니다.
(예: 'A' → 65)

🖨️ chr(숫자)

3D 프린터
설계도 번호(숫자)를 넣으면 해당 문자를 찍어냅니다.
(예: 65 → 'A')

1. 유니코드(Unicode)란?

전 세계의 모든 문자를 컴퓨터에서 일관되게 표현하기 위해 만든 '문자들의 지도'입니다.

영어, 한글, 특수기호, 이모지(😀)까지 모든 문자는 자신만의 고유 번호를 가지고 있습니다.

# 문자는 사실 숫자입니다!
'A' = 65
'a' = 97
'0' = 48
'가' = 44032

2. 문자를 숫자로: ord()

ord() 함수는 문자 하나를 입력받아 그 문자의 유니코드 정수값(Ordinal)을 반환합니다.

print(ord('A'))       # 대문자 A는 65번
print(ord('a'))       # 소문자 a는 97번
print(ord(' '))       # 공백(space)도 번호가 있음 (32번)
print(ord('가'))      # 한글 '가'는 44032번
65 97 32 44032
💡 잠깐! 대소문자의 비밀

소문자 'a'(97)와 대문자 'A'(65)의 차이를 계산해 볼까요?
97 - 65 = 32 입니다.

놀랍게도 'b' - 'B', 'z' - 'Z' 모두 32만큼 차이가 납니다.
이 규칙(32)을 알면 대소문자를 서로 변환할 때 유용하게 쓸 수 있습니다!

✏️ [중간 점검] 코드 예측

print(ord('B'))의 결과는 무엇일까요?
(힌트: 'A'가 65입니다.)

정답 확인하기
정답: 66
해설: 유니코드에서 알파벳은 순서대로 번호가 매겨집니다. A(65) 다음 B는 66입니다.

3. 숫자를 문자로: chr()

chr() 함수는 정수(Character Code)를 입력받아 그 번호에 해당하는 문자를 반환합니다.

print(chr(65))        # 65번은 'A'
print(chr(66))        # 66번은 'B'
print(chr(97))        # 97번은 'a'
print(chr(44032))     # 44032번은 '가'
A B a 가

4. 응용: 알파벳과 암호

알파벳이 숫자로 되어 있다는 점을 이용하면 재미있는 일들을 할 수 있습니다.

4-1. 알파벳 순서대로 출력하기

for 문으로 번호를 하나씩 증가시키며 출력합니다.

# 65(A)부터 70(F) 전까지 반복
for i in range(65, 70):
    print(chr(i), end=" ")
A B C D E

4-2. 암호 만들기 (시저 암호)

문자를 다른 문자로 바꾸는 가장 고전적인 방법이 시저 암호(Caesar Cipher)입니다.
유니코드 번호에 +1을 하면 바로 다음 글자로 바뀌는 원리를 이용합니다. 

origin = 'A'
secret = chr(ord(origin) + 1)  # 65 + 1 = 66 ('B')
print(secret)
B

✏️ [중간 점검] 소문자 출력

'a'부터 'c'까지 소문자를 출력하려면 range를 어떻게 설정해야 할까요?
(a=97)

정답 확인하기
정답: range(97, 100)
해설: 97(a), 98(b), 99(c)까지 출력해야 하므로 끝값은 100이 되어야 합니다.

🚀 실전 코딩 챌린지

배운 내용을 활용해 비밀 코드를 만들어 볼까요?

Mission 1
다음 문자 찾기

사용자에게 문자 한 개를 입력받아, 알파벳 순서상 바로 다음 문자를 출력하세요.
(예: A → B, c → d)

입력: A 결과: B
정답 코드 확인하기 
char = input("문자 입력: ")
code = ord(char)       # 숫자로 변환
next_char = chr(code + 1) # 1 더해서 다시 문자로
print(next_char)
Mission 2
비밀 암호 만들기 (시저 암호)

문자열 "LOVE"의 각 글자를 +1만큼 이동시켜 암호로 만드세요.
(L→M, O→P, V→W, E→F)

원본: LOVE 암호: MPWF
정답 코드 확인하기 
word = "LOVE"
for char in word:
    # encrypted 대신 쉬운 변수명 new_char 사용
    new_char = chr(ord(char) + 1)
    print(new_char, end="")
Mission 3
알파벳 Z까지 출력하기

ord()와 chr()을 사용하여 'A'부터 'Z'까지 한 줄에 출력하는 코드를 작성하세요.

A B C ... X Y Z
정답 코드 확인하기 
# ord('Z')는 90입니다. 끝값은 91이어야 함
for i in range(ord('A'), ord('Z') + 1):
    print(chr(i), end=" ")
Mission 4
숫자 판별기

입력받은 문자가 숫자('0'~'9')인지 판별하는 코드를 작성하세요.
(문자열 메서드 isdigit()을 쓰지 않고 유니코드 범위로 확인해보세요!)

입력: 5 결과: 숫자입니다. 입력: a 결과: 숫자가 아닙니다.
정답 코드 확인하기 
char = input("한 글자 입력: ")
code = ord(char)

# '0'의 코드는 48, '9'는 57
if ord('0') <= code <= ord('9'):
    print("숫자입니다.")
else:
    print("숫자가 아닙니다.")
