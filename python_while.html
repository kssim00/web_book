1. While 문의 기본 구조

while은 '~하는 동안'이라는 뜻입니다. 조건이 맞으면 계속 실행하고, 틀리면 즉시 종료합니다. while문은 3가지를 항상 직접 써줘야 합니다.

① 시작값

i = 1

반복을 어디서 시작할지

② 조건식

while i <= 3:

언제까지 반복할지

③ 증가식

i += 1

매 반복마다 변수를 바꿔줌

③ 증가식을 빠뜨리면 조건이 영원히 True가 되어 무한루프가 발생합니다. ① 시작값을 안 쓰면 NameError가 발생합니다.

i = 1                  # ① 시작값 선언 (필수!)

while i <= 3:         # ② 조건식 (참일 동안 반복)
    print(i)
    i += 1             # ③ 증가식 (없으면 무한루프!)

2. 데이터의 끝을 알리는 보초값(Sentinel)

반복 횟수를 미리 알 수 없을 때, 특별한 값을 입력하면 반복을 멈추도록 약속하는 것이 보초값입니다. 보초값은 실제 데이터로 절대 나올 수 없는 값으로 정해야 합니다.

반복 횟수를 아는 경우

"1부터 10까지 출력해" → 횟수가 정해져 있어서 while 조건으로 바로 제어 가능

반복 횟수를 모르는 경우 — 보초값 필요!

"점수를 계속 입력받아" → 몇 개 입력할지 모름. 특별한 값을 입력하면 멈추도록 약속해야 함

score = int(input("점수 입력(-1 종료): "))   # 첫 입력

while score != -1:                          # -1이 아닌 동안 반복
    print("입력된 점수:", score)
    score = int(input("점수 입력(-1 종료): ")) # 다음 입력

print("입력 종료!")

💡 왜 while 전에 먼저 input을 받나요? — while 조건에서 score를 비교하려면 score가 이미 있어야 해요. 처음부터 값이 없으면 오류가 납니다.

3. 흐름 제어: Break & Continue

반복 중간에 흐름을 제어할 수 있습니다. break는 반복을 즉시 끝내고, continue는 현재 순서를 건너뜁니다.

Break

반복문을 완전히 끝냅니다. 조건과 상관없이 즉시 탈출합니다.

결과: 1 2

Continue

이번 회만 건너뛰고, 다음 조건 확인으로 돌아갑니다.

결과: 1 2 4 5

3을 만나는 순간 반복이 완전히 끝납니다.

i = 1
while i <= 5:
    if i == 3:
        break
    print(i)
    i += 1
print("반복 끝")

3만 건너뛰고 반복은 계속됩니다.

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
print("반복 끝")

한눈에 비교 — i == 3일 때 무슨 일이 생길까?

키워드 i == 3 일 때 출력 결과
break 반복문 자체를 완전히 끝냄 1 2 반복 끝
continue 3만 건너뛰고 반복은 계속 1 2 4 5 반복 끝

4. 무한 루프 (Infinite Loop)

while True:로 일부러 무한 반복을 만들고, 원하는 조건이 되면 break로 탈출하는 패턴입니다. 몇 번 반복할지 미리 알 수 없을 때 사용합니다.

while 조건식

끝나는 시점이 처음부터 정해진 경우

i = 1
while i <= 10:
    print(i)
    i += 1

while True + break

끝나는 시점을 실행 중에 판단하는 경우

while True:
    pw = input()
    if pw == "1234":
        break
while True:
    code = input("암호를 입력하세요: ")
    if code == "1234":
        print("정답! 탈출!")
        break              # 맞으면 반복 종료
    print("틀렸습니다. 다시 입력하세요.")
암호를 입력하세요: hello
틀렸습니다. 다시 입력하세요.
암호를 입력하세요: 1234
정답! 탈출!

5. while — else

while문은 else와 짝을 이룰 수 있습니다. 조건이 False가 되어 정상 종료되면 else 블록이 실행됩니다.

break로 탈출하면 else가 실행되지 않습니다. 이 차이가 핵심입니다!

count = 0
while count < 3:
    print(count, '은 3보다 작은 수')
    count = count + 1
else:
    print(count, '이다! stop!!')
0 은 3보다 작은 수
1 은 3보다 작은 수
2 은 3보다 작은 수
3 이다! stop!!

Example — break 유무에 따른 else 실행 차이

아래 두 코드를 비교하면서 else가 언제 실행되고, 언제 실행되지 않는지 확인해봅시다.

조건이 False가 되어 정상 종료 → else 실행

i = 1
while i <= 3:
    print(i)
    i += 1
else:
    print("정상 종료")
