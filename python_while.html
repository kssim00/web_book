while문의 기본 구조

조건이 True인 동안 계속 반복하고, False가 되면 즉시 종료합니다.

while 조건:
    실행할 코드

1. while문의 기본 구조

아래 코드들을 보세요. for문과 무엇이 다른지 생각해보세요.

i = 1
while i <= 3:
    print(i)
    i += 1
→
1 2 3

💡 알 수 있는 것

while은 조건이 참(True)인 동안 계속 반복합니다.
조건이 거짓(False)이 되는 순간 반복이 끝납니다.

while문은 3가지를 반드시 직접 작성해야 합니다.

① 시작값

i = 1

반복을 어디서 시작할지

② 조건식

while i <= 3:

언제까지 반복할지

③ 증가식

i += 1

매 반복마다 변수를 바꿔줌

⚠️ ③ 증가식을 빠뜨리면 조건이 영원히 True가 되어 무한루프가 발생합니다.
⚠️ ① 시작값을 선언하지 않으면 NameError가 발생합니다.

✏️ Example — 1부터 5까지 출력하기

3가지 요소(시작값, 조건식, 증가식)를 모두 직접 써서 1부터 5까지 출력해봅시다.

2. 데이터의 끝을 알리는 보초값(Sentinel)

반복 횟수를 아는 경우

"1부터 10까지 출력해" → 횟수가 정해져 있어서 while 조건으로 바로 제어 가능

반복 횟수를 모르는 경우 → 보초값 필요!

"점수를 계속 입력받아" → 몇 개 입력할지 모름. 특별한 값을 입력하면 멈추도록 약속해야 함

아래 코드는 몇 번 반복할지 미리 정해져 있지 않습니다. 어떻게 동작할까요?

score = int(input("점수 입력(-1 종료): "))

while score != -1:
    print("입력된 점수:", score)
    score = int(input("점수 입력(-1 종료): "))

print("입력 종료!")
→
입력된 점수: 90 입력된 점수: 75 입력 종료!

💡 알 수 있는 것

반복 횟수를 미리 알 수 없을 때, 특별한 값을 입력하면 반복을 멈추도록 약속합니다.
이 특별한 값을 보초값(Sentinel)이라고 합니다.
보초값은 실제 데이터로 절대 나올 수 없는 값으로 정해야 합니다.

보초값을 선택하는 원칙을 정리하면 다음과 같습니다.

상황실제 데이터 범위적합한 보초값
시험 점수 입력0 ~ 100-1 (범위 밖)
주사위 숫자 입력1 ~ 60 또는 7 (범위 밖)
상품 가격 입력0 이상-1 (음수)
⚠️ 왜 while 전에 먼저 input을 받나요?
while 조건에서 score를 비교하려면 score가 이미 선언되어 있어야 합니다. 처음부터 값이 없으면 NameError가 납니다.

3. 흐름 제어: break & continue

두 코드의 결과를 비교해보세요. i == 3일 때 어떤 차이가 있나요?

break — 3을 만나는 순간 반복이 완전히 끝납니다.

i = 1
while i <= 5:
    if i == 3:
        break
    print(i)
    i += 1
print("반복 끝")
1 2 반복 끝

continue — 3만 건너뛰고 반복은 계속됩니다.

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
print("반복 끝")
1 2 4 5 반복 끝

💡 알 수 있는 것

break: 반복문을 완전히 종료합니다.
continue: 이번 회만 건너뛰고 다음 반복으로 돌아갑니다.

키워드i == 3 일 때출력 결과
break반복문 자체를 완전히 끝냄1 2 반복 끝
continue3만 건너뛰고 반복은 계속1 2 4 5 반복 끝

4. 무한 루프 (Infinite Loop)

아래 코드는 어떻게 종료될까요?

while True:
    code = input("암호를 입력하세요: ")
    if code == "1234":
        print("정답! 탈출!")
        break
    print("틀렸습니다. 다시 입력하세요.")
→
틀렸습니다. 다시 입력하세요. 틀렸습니다. 다시 입력하세요. 정답! 탈출!

💡 알 수 있는 것

while True:로 일부러 무한 반복을 만들고, 원하는 조건이 되면 break로 탈출하는 패턴입니다.
몇 번 반복할지 미리 알 수 없을 때 주로 사용합니다.

방식언제 사용?예시
while 조건식끝나는 시점이 처음부터 정해진 경우1부터 10까지 반복
while True + break끝나는 시점을 실행 중에 판단하는 경우비밀번호가 맞을 때까지

while 조건식

끝나는 시점이 처음부터 정해진 경우

i = 1
while i <= 10:
    print(i)
    i += 1

while True + break

끝나는 시점을 실행 중에 판단하는 경우

while True:
    pw = input()
    if pw == "1234":
        break

5. while — else

두 코드를 비교해보세요. else가 언제 실행되고, 언제 실행되지 않나요?

조건이 False가 되어 정상 종료 → else 실행

i = 1
while i <= 3:
    print(i)
    i += 1
else:
    print("정상 종료")
1 2 3 정상 종료

break로 탈출 → else 실행되지 않음

i = 1
while i <= 3:
    if i == 2: break
    print(i)
    i += 1
else:
    print("정상 종료")
1

💡 알 수 있는 것

else는 while 조건이 False가 되어 정상 종료될 때 실행됩니다.
break로 탈출하면 else가 실행되지 않습니다. 이 차이가 핵심입니다!
