파이썬 While 문 완전 정복

조건이 '참'인 동안은 절대 멈추지 않는 무한 동력!

🚦 검문소 (조건 확인)

코드가 실행되기 전에
"조건이 True인가?"를 먼저 검사합니다.
False라면 문을 열어주지 않아요.

🏃 런닝머신 (반복 수행)

정지 버튼(조건 변경)을 누르기 전까지
끝없이 계속 달립니다.
멈추는 코드를 넣지 않으면 영원히 돕니다!

1. While 문의 기본 구조

while은 '~하는 동안'이라는 뜻입니다. 조건이 맞으면 계속 실행하고, 틀리면 즉시 종료합니다.

i = 0                 # 1. 시작값

while i < 3:        # 2. 조건식 (참일 동안 반복)
    i = i + 1         # 3. 증감식 (필수!)
    print(f"현재 숫자는 {i}")
현재 숫자는 1
현재 숫자는 2
현재 숫자는 3

2. 데이터의 끝을 알리는 '보초값(Sentinel)'

반복 횟수를 정확히 모를 때, "이 값이 나오면 멈춰라!"라고 약속한 값을 사용합니다.
데이터 입력 프로그램에서 -1, 0, "quit" 등을 종료 신호로 사용합니다. 

score = 0
# -1이 입력될 때까지 계속 반복
while score != -1:
    score = int(input("점수 입력(-1 종료): "))
    print(f"입력된 점수: {score}")
    
print("프로그램 종료")

✏️ [중간 점검] 보초값 찾기

다음 코드에서 반복문을 종료시키는 '보초값'은 무엇인가요?

pw = ""
while pw != "1234":
    pw = input("비밀번호: ")
정답 확인하기
정답: "1234"
설명: 사용자가 "1234"를 입력하는 순간 조건이 False가 되어 반복이 종료됩니다.

3. 흐름 제어: Break vs Continue

반복문을 자유자재로 다루기 위한 두 가지 핵심 키워드를 비교해봅시다. (가장 중요! ⭐)

🛑 Break

반복문을 박차고 나갑니다.
뒤에 남은 코드는 실행되지 않고, 반복 자체가 완전 종료됩니다.

i = 0
while i < 5:
    i += 1
    if i == 3:
        break
    print(i, end=" ")
결과: 1 2
⏭️ Continue

이번 순서만 건너뛰고,
다음 순서(조건 확인)로 돌아갑니다. 반복은 계속됩니다.

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i, end=" ")
결과: 1 2 4 5
# [응용] Continue를 활용한 369 게임 (3의 배수 건너뛰기)
num = 0

while num < 10:
    num += 1

    if num % 3 == 0:
        print("짝!", end=" ")
        continue

    print(num, end=" ")

4. 무한 루프 (Infinite Loop)

조건을 항상 True로 설정하면 영원히 멈추지 않는 프로그램이 됩니다.
반드시 break를 이용해 탈출 구멍을 만들어주어야 합니다. 

# 암호 입력기 (맞출 때까지 무한 반복)
while True:
    code = input("암호를 대시오: ")
    
    if code == "열려라참깨":
        print("문이 열렸습니다!")
        break  # 탈출!
        
    print("틀렸습니다. 다시 시도하세요.")

Mission 1
NASA 로켓 발사 카운트다운

사용자에게 시작 숫자(예: 5)를 입력받아 1까지 카운트다운을 출력하고,
마지막에 "🚀 발사!"를 출력하세요. (변수 값을 1씩 줄여나가세요)

입력: 5
5... 4... 3... 2... 1... 🚀 발사!
정답 코드 확인하기 
n = int(input("시작 숫자: "))

while n > 0:
    print(f"{n}...", end=" ")
    n = n - 1

print("🚀 발사!")
Mission 2
폭탄 해체 코드 입력기 (Break)

무한 루프를 이용해 코드를 계속 입력받습니다.
입력값이 "해제"면 "성공!"을 출력하고 반복을 종료하고,
아니면 "남은 시간: X초" 메시지를 계속 띄우세요.

정답 코드 확인하기 
while True:
    code = input("코드 입력: ")
    if code == "해제":
        print("폭탄 해체 성공! 🎉")
        break
    print("⏳ 시간이 흐르고 있습니다...")
Mission 3
세계 여행: 나라 수도 맞추기

[1.한국 2.미국 3.일본] 메뉴를 보여주고 번호를 입력받습니다.
해당하는 수도를 출력(예: 한국->서울)하고, 없는 번호나 종료 신호가 들어오면 종료하세요.

메뉴: [1.한국 2.미국 3.일본]
선택: 1 -> 서울
선택: 2 -> 워싱턴
선택: 5 -> 알 수 없는 나라입니다. 종료.
정답 코드 확인하기 
while True:
    print("[1.한국 2.미국 3.일본]")
    choice = int(input("번호 선택: "))

    if choice == 1:
        print("서울")
    elif choice == 2:
        print("워싱턴")
    elif choice == 3:
        print("도쿄")
    else:
        print("종료합니다.")
        break
Mission 4
누적 합계 구하기 (0 입력 시 종료)

숫자를 계속 입력받아 합계를 더해나갑니다.
0을 입력하면 입력을 멈추고 최종 합계를 출력하세요.

숫자: 10
숫자: 20
숫자: 0
결과: 30
정답 코드 확인하기 
total = 0
while True:
    num = int(input("숫자: "))
    if num == 0:
        break
    total += num

print(f"최종 합계: {total}")
Mission 5
양수/음수 판별기

숫자를 입력받아 양수인지 음수인지 판별하세요.
0이 입력되면 "프로그램 종료"를 출력하고 멈추세요.

입력: 5 -> 양수입니다
입력: -3 -> 음수입니다
입력: 0 -> 프로그램 종료
정답 코드 확인하기 
while True:
    num = int(input("숫자 입력: "))
    
    if num == 0:
        print("프로그램 종료")
        break
    elif num > 0:
        print("양수입니다")
    else:
        print("음수입니다")
Mission 6
제곱수 출력하기

숫자 N을 입력받아, 1부터 N까지의 제곱수를 모두 출력하세요.
(예: 3 입력 시 -> 1의 제곱은 1, 2의 제곱은 4, 3의 제곱은 9)

정답 코드 확인하기 
limit = int(input("숫자 입력: "))
n = 1

while n <= limit:
    print(f"{n}의 제곱은 {n*n}")
    n += 1
