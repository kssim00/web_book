1. 함수의 기본 구조

함수는 def 키워드로 시작합니다. 'define(정의하다)'의 약자입니다.

# 1. 함수 정의 (만들기)
def hello():
    print("안녕하세요!")

# 2. 함수 호출 (사용하기)
hello()

2. 매개변수(Parameter)와 인수(Argument)

함수에 재료(값)를 전달할 수 있습니다.
자판기에 '투입구'가 있어야 동전을 넣을 수 있는 것과 같습니다. 

# name이라는 '투입구(빈 칸)'를 뚫어놓음
def greet(name):
    print(f"{name}님, 환영합니다!")

# 투입구에 '실제 값'을 넣어줌
greet("철수")

3. 리턴(Return) vs 프린트(Print)

가장 많이 헷갈리는 개념입니다. Return은 값을 '주는' 것이고, Print는 '보여주는' 것입니다.

↩️ Return

값을 계산해서 나에게 돌려줍니다.
변수에 저장할 수 있습니다.

def plus(a, b):
    return a + b  # 값 반환

x = plus(1, 2)
print(x)  # 3 출력됨
📺 Print

화면에 글자를 보여주기만 합니다.
변수에 저장할 수 없습니다. (None)

def show(a, b):
    print(a + b) # 출력만 함

x = show(1, 2)
print(x)  # None (비어있음)

4. 변수의 범위 (Scope)

함수 안에서 만든 변수는 함수 안에서만 살아있습니다. 이것을 '지역 변수'라고 합니다.
반대로 어디서든 쓸 수 있는 변수는 '전역 변수'라고 합니다.

🏠 우리 집 (함수 안)

지역 변수 (Local)
내 방에 있는 '인형'은
집 밖에서 볼 수 없어요.
함수가 끝나면 사라집니다.

🏙️ 광장 (함수 밖)

전역 변수 (Global)
광장에 있는 '동상'은
집 안에서도 보여요.
프로그램 내내 유지됩니다.

toy = "동상"      # 전역 변수

def my_room():
    doll = "인형" # 지역 변수
    print(toy)    # 집 안에서 동상은 보임 (OK)
    print(doll)   # 내 인형도 보임 (OK)

my_room()
# print(doll)     # 에러! 밖에서는 남의 집 인형 안 보임

5. 알아두면 좋은 고급 기능

함수를 더 똑똑하게 사용하는 두 가지 방법입니다.

① 기본값 매개변수 (Default Argument)

값을 입력하지 않았을 때 자동으로 들어갈 '기본값'을 정해줄 수 있습니다. 단, 기본값은 항상 맨 뒤에 적어야 합니다!

def order(menu, count=1):  # count를 안 넣으면 1로 간주
    print(f"{menu} {count}개 주문")

order("콜라")      # "콜라 1개 주문"
order("사이다", 3) # "사이다 3개 주문"

② 키워드 인수 (Keyword Argument)

순서가 헷갈릴 때는 이름을 직접 지정해서 값을 넣을 수 있습니다. 순서가 바뀌어도 상관없습니다.

def profile(name, age):
    print(name, age)

# 순서를 바꿔도 이름(key)을 맞춰주면 OK!
profile(age=20, name="영희")

6. 함수의 확장 기술 (심화)

퀴즈를 풀기 위해 꼭 알아야 할 숨겨진 마법들입니다. (Q14, Q18, Q20 관련)

① 가변 인자 (*args)

입력값이 몇 개인지 모를 때 사용합니다. 값들을 튜플(Tuple)로 묶어서 받습니다.

def many_nums(*args):
    print(args)  # (1, 2, 3) 튜플 형태

many_nums(1, 2, 3)

② Global 키워드

함수 안에서 바깥의 전역 변수를 수정하고 싶을 때 global을 씁니다.

count = 0
def increase():
    global count  # 바깥에 있는 count를 쓸 거야!
    count += 1

③ 다중 반환 (Tuple Return)

파이썬은 return a, b 처럼 여러 값을 한 번에 돌려줄 수 있습니다. 이때도 튜플로 묶여서 나옵니다.

def get_two():
    return 1, 2  # (1, 2) 반환

Mission 1
절대값 구하기 함수

숫자 하나를 입력받아, 양수면 그대로, 음수면 양수로 바꿔서 반환(return)하는 함수 my_abs(n)을 만드세요.
(내장함수 abs 사용 금지)

print(my_abs(-5)) -> 5
print(my_abs(10)) -> 10
정답 코드 확인하기 
def my_abs(n):
    if n < 0:
        return -n
    else:
        return n
Mission 2
짝수 판별기 (True/False)

숫자를 하나 입력받아 짝수면 True, 홀수면 False를 반환하는 함수 is_even(n)을 만드세요.

print(is_even(4)) -> True
print(is_even(7)) -> False
정답 코드 확인하기 
def is_even(n):
    # Q15 개념: 불리언(True/False)도 반환 가능!
    if n % 2 == 0:
        return True
    else:
        return False
Mission 3
합계 구하기 함수

숫자 N을 입력받아 1부터 N까지의 합을 구해서 반환하는 함수 get_sum(n)을 작성하세요.

result = get_sum(10)
print(result) -> 55
정답 코드 확인하기 
def get_sum(n):
    total = 0
    for i in range(1, n + 1):
        total += i
    return total
Mission 4
계산기 함수 (사칙연산)

숫자 두 개와 연산자(+, -, *, /)를 입력받아 결과를 반환하는 calc(a, b, op) 함수를 만드세요.

print(calc(10, 20, '+')) -> 30
print(calc(10, 2, '/')) -> 5.0
정답 코드 확인하기 
def calc(a, b, op):
    if op == '+':
        return a + b
    elif op == '-':
        return a - b
    elif op == '*':
        return a * b
    elif op == '/':
        return a / b
    else:
        return "잘못된 연산자"
Mission 5
리스트 평균 구하기

숫자들이 들어있는 리스트를 인수로 받아, 평균값을 반환하는 get_avg(num_list) 함수를 작성하세요.

scores = [80, 90, 100]
print(get_avg(scores)) -> 90.0
정답 코드 확인하기 
def get_avg(num_list):
    if len(num_list) == 0:
        return 0
    
    total = sum(num_list)
    avg = total / len(num_list)
    return avg

별 찍기 함수 (Void 함수)

줄 수(n)를 입력받아, n줄짜리 삼각형 별을 출력(Print)하는 함수 print_stars(n)을 만드세요.

print_stars(3) 실행 시:
*
**
***
정답 코드 확인하기 
def print_stars(n):
    for i in range(1, n + 1):
        print("*" * i)
