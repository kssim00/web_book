파이썬 딕셔너리(Dictionary) 완전 정복

Key(열쇠)로 Value(값)을 즉시 찾아내는 초고속 데이터 저장소

🏷️ 대형마트 바코드

바코드(Key)를 찍으면
즉시 가격 정보(Value)가 나옵니다.
몇 번째 물건인지 찾을 필요가 없죠.

🗄️ 사물함 (Locker)

각 칸마다 이름표(Key)가 붙어 있습니다.
"철수 사물함 열어!" 하면
그 안의 물건(Value)을 바로 꺼냅니다.

1. 딕셔너리의 기본 구조

딕셔너리는 중괄호 {}를 사용하며, Key : Value 쌍으로 이루어집니다.
리스트가 순서(Index)로 데이터를 관리한다면, 딕셔너리는 이름표(Key)로 관리합니다. 

# 1. 딕셔너리 생성 (Key : Value)
hero = {
    "이름": "아이언맨",
    "슈트": "MK-85",
    "파워": 100
}

# 2. 값 꺼내기 (Key를 사용)
print(hero["이름"])  # 아이언맨
print(hero["파워"])  # 100

[중간 점검] 문법 체크

다음 중 올바른 딕셔너리 형태는 무엇일까요?

1. {'name', 'Kim', 'age', 20}
2.['name': 'Kim', 'age': 20]
3. {'name': 'Kim', 'age': 20}
4. ('name': 'Kim', 'age': 20)
정답 확인하기
정답: 3번
딕셔너리는 중괄호 {}를 사용하며, Key와 Value 사이에는 콜론(:), 각 쌍 사이에는 쉼표(,)가 있어야 합니다.

2. 리스트(List) vs 딕셔너리(Dict)

언제 무엇을 써야 할까요? 데이터를 찾는 방식이 완전히 다릅니다.

📋 리스트 (List)

번호(Index)로 관리합니다.
순서가 중요할 때 사용합니다.

scores = [90, 85]
print(scores[1]) 
# 85 (누구 점수인지 모름)
🔑 딕셔너리 (Dict)

이름(Key)으로 관리합니다.
의미를 부여해서 찾을 때 사용합니다.

scores = {"철수": 90, "영희": 85}
print(scores["영희"]) 
# 85 (영희 점수 확인!)

3. 데이터 조작하기 (CRUD)

딕셔너리는 데이터를 자유롭게 추가, 수정, 삭제할 수 있습니다.

구분 코드 예시 설명
추가/수정 data['key'] = value Key가 없으면 추가, 있으면 수정됩니다.
삭제 del data['key'] 해당 Key와 쌍을 이루는 값을 삭제합니다.
초기화 data.clear() 모든 내용을 지워서 빈 딕셔너리로 만듭니다. 
my_dict = {"A": 10}

# 1. 추가 (B가 없으므로 추가됨)
my_dict["B"] = 20  

# 2. 수정 (A가 있으므로 값만 바뀜)
my_dict["A"] = 99  

# 3. 삭제
del my_dict["B"]   

print(my_dict) # {'A': 99}

4. ★ 안전하게 조회하기 (get 메서드)

베테랑 개발자가 되기 위한 필수 습관! 없는 키를 찾으면 에러가 납니다.

❌ 위험한 방식: [ ]

없는 키를 넣으면 KeyError가 발생하며
프로그램이 멈춥니다.

info = {"name": "Sonic"}
# print(info["age"]) 
# -> 에러 발생! 💥
✅ 안전한 방식: .get()

없는 키를 넣어도 에러 대신 None을 반환합니다.

info = {"name": "Sonic"}
print(info.get("age")) 
# None (프로그램 안 멈춤)

[중간 점검] get()의 활용

info.get('age', 0) 처럼 쓰면, 값이 없을 때 None 대신 0을 반환합니다. 이를 '기본값'이라고 합니다.

실행 결과 예측하기
키 'age'가 없으므로, 기본값으로 설정한 0이 반환됩니다.
이를 활용하면 카운팅(투표) 프로그램을 아주 쉽게 짤 수 있습니다.

5. 딕셔너리 분해와 반복 (Loop)

딕셔너리 안에 있는 내용물을 반복문(for)으로 꺼내는 방법입니다.

① Key만 꺼내기 (기본)

ages = {"철수": 10, "영희": 12}
for key in ages:  # 또는 ages.keys()
    print(key)    # 철수, 영희 출력

② Value(값)만 꺼내기

for val in ages.values():
    print(val)    # 10, 12 출력

③ 쌍(Pair)으로 꺼내기 (가장 많이 씀)

items()를 사용하면 키와 값을 동시에 꺼낼 수 있습니다.

for name, age in ages.items():
    print(f"{name}는 {age}살")

6. 주의! Key가 될 수 없는 것들

딕셔너리의 열쇠(Key)는 변하지 않는 값(Immutable)이어야 합니다.

  • ✅ 가능: 정수, 실수, 문자열, 튜플(Tuple)
  • ❌ 불가능: 리스트(List), 딕셔너리(Dict) - 내용이 바뀔 수 있기 때문!
# 에러 발생하는 코드
my_dict = {}
my_list = [1, 2]
# my_dict[my_list] = "값" 
# TypeError: unhashable type: 'list'

Mission 1
단어 투표 집계 (Word Counter)

과일 이름이 적힌 리스트가 있습니다. 각 과일이 몇 번 나왔는지 세어서 딕셔너리로 만드세요.
입력: ['apple', 'banana', 'apple', 'apple']
목표 출력: {'apple': 3, 'banana': 1}

정답 코드 확인하기 
fruits =['apple', 'banana', 'apple', 'apple']
counts = {}

for f in fruits:
    if f in counts:
        counts[f] += 1
    else:
        counts[f] = 1

print(counts)
Mission 2
성적 등급 변환기

학생들의 점수가 담긴 딕셔너리가 있습니다. 80점 이상인 학생만 골라내어
새로운 딕셔너리 passed_students를 만드세요.
입력: scores = {'철수': 70, '영희': 90, '민수': 85}

정답 코드 확인하기 
scores = {'철수': 70, '영희': 90, '민수': 85}
passed_students = {}

for name, score in scores.items():
    if score >= 80:
        passed_students[name] = score

print(passed_students)
Mission 3
카페 키오스크 (가격 계산)

메뉴판 딕셔너리가 있습니다. 주문 리스트를 받아 총 금액을 계산하는 함수를 작성하세요.
menu = {'아아': 2000, '라떼': 3000}
order =['아아', '아아', '라떼'] -> 결과: 7000

정답 코드 확인하기 
menu = {'아아': 2000, '라떼': 3000}
order =['아아', '아아', '라떼']

total = 0
for item in order:
    total += menu[item]

print(f"총 금액: {total}원")
